 Sales Performance Dashboard
Data Analytics project using Excel and Power BI

  Project Workflow
1. Collected raw sales data with missing and inconsistent values
2. Cleaned data in Excel:
   - Filled missing customer names
   - Handled blank quantity and profit values
   - Managed negative values
   - Standardized column names
3. Created Sales Level categories
4. Converted data into Excel tables
5. Imported cleaned data into Power BI
6. Built visuals:
   - Daily sales performance
   - Total sales by product name(descending)
   - Sales by category
   - Monthly trend analysis
8. Created interactive dashboard
9. Uploaded files to GitHub with documentation
